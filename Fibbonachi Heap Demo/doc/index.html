LinkedNode Class Documentation

A class that represents a linked node in a doubly linked list. Each node stores an integer value and has two neighbors, the previous (_lNeighbor) and the next (_rNeighbor).

Constructor Summary

Constructor Description
LinkedNode(value=None, parent=None) Constructs a new LinkedNode object.

Method Summary

Method Description Returns
getPrev() Returns the left neighbor of the node. a LinkedNode object that represents the left neighbor
getNext() Returns the right neighbor of the node. a LinkedNode object that represents the right neighbor
isHead() Returns True if the node is the head of the list (i.e., has no left neighbor), False otherwise. a boolean value indicating whether the node is the head of the list
isTail() Returns True if the node is the tail of the list (i.e., has no right neighbor), False otherwise. a boolean value indicating whether the node is the tail of the list
getValue() Returns the value stored in the node. an integer value representing the node's stored value
