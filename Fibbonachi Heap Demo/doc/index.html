LinkedNode

Represents a node in a linked list.

Constructor Summary

Constructor Description
LinkedNode(value=None,parent=None) Creates a new LinkedNode object with the given value and parent node.

Method Summary

Method Description
getPrev() Returns the previous node in the linked list.
getNext() Returns the next node in the linked list.
isHead() Returns true if this node is the head of the linked list.
isTail() Returns true if this node is the tail of the linked list.
getValue() Returns the value of this node.
